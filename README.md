Bem-vindo ao repositório oficial de distribuição do **INPKT Tools**. 

Este repositório é utilizado **exclusivamente para hospedar as versões compiladas (binários e instaladores)** do nosso aplicativo.

> ⚠️ **Atenção:** O INPKT Tools possui arquitetura de código-fonte **proprietária e fechada**. A hospedagem neste repositório do GitHub existe apenas para facilitar o acesso público, ágil e transparente aos instaladores gratuitos. Este projeto **não é Open Source**.

## 📥 Como Baixar

Acesse a aba [**Releases**](../../releases) no painel lateral direito deste repositório para encontrar a versão mais recente.
Selecione o arquivo ideal para o seu sistema:

- 🪟 **Windows:** Baixe o `inpktools-amd64-installer.exe` ou `.msi`.
- 🐧 **Linux:** Escolha entre `.deb`, `.AppImage` ou instale via Snap Store.
